================================================================================
                             TROUBLESHOOTING
================================================================================

ERROR: "Neural sync error during object detection"

SOLUTION:
1. Verify API key: cat .env
2. Check balance: https://platform.minimax.io/user-center/payment/balance
3. Try different image (JPEG/PNG, < 5MB)
4. Check browser console (F12)

ERROR: Diagram/Video timeout

SOLUTION:
1. Wait up to 90 seconds
2. Check Minimax API status
3. Verify sufficient balance
4. Try simpler prompts

ERROR: Chat not responding

SOLUTION:
1. Check API key in .env
2. Verify blueprint generated successfully
3. Check network tab (F12 → Network)
